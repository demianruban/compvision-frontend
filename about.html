About our project

The project idea was initially created with brainstorm. We sat down at the "round table" of discord voice chat and started to extract ideas from our heads. Our list contained other ideas, such as portfolio website, weather logger. In the end, however, we unanimously chose the first idea - Computer Vision API frontend. One of the developers already was familiar with public api repository on github so we had a list where to choose from. We chose image labeler API by inferdo. Our idea was to make this project simple and don't use any huge frameworks.
